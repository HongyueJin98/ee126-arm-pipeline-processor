# ee126-arm-pipeline-processor
# 5-Stage MIPS Pipeline Processor

A VHDL implementation of a classic five-stage pipelined MIPS processor developed for EE126 Computer Engineering.

The project was implemented and verified using VHDL on Linux with the QuestaSim / ModelSim simulation environment. The design demonstrates fundamental concepts in computer architecture, including instruction execution, pipelining, control logic, and hardware simulation.

---

## Overview

This project implements a five-stage MIPS pipeline processor consisting of:

* Instruction Fetch (IF)
* Instruction Decode (ID)
* Execute (EX)
* Memory Access (MEM)
* Write Back (WB)

The processor was built incrementally throughout the course and integrates the core components required for instruction execution and pipelined operation.

---

## Features

* Five-stage pipelined processor architecture
* VHDL-based hardware implementation
* Arithmetic Logic Unit (ALU)
* Register File
* Control Unit
* Instruction Memory
